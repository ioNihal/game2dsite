Stickman Fighter 2D

About the game

Stickman Fighter 2D is a fast paced, endlessly replayable 1v1 fighting game built on HTML5 Canvas and vanilla JavaScript. It's a solo passion project aimed at exploring:

  • Responsive input on desktop & mobile
  • Simple AI with adjustable difficulty
  • Modular animation & physics systems
  • Integrated sound effects & music
  • Clean UI with menu, pause, and settings screens

Controlls

Use A or ← and D or → for Movements

Use W or ↑ for Jumping

Use J for Attacking

Use K for Blocking

Use ESC or Pause Button for Blocking
